In order to gain a general sense of the data we have obtained, let us plot the data. Since each independent variable has multiple trials and therefore has multiple dependent data points, we use boxplots:

boxplot(time_10_periods ~ length_cm, data = data,
    main = "Effect of Length on Time",
    xlab = "Length (cm)",
    ylab = "Time (10 periods)")

The data is roughly linear. So,

model <- lm(time_10_periods ~ length_cm, data = data)
